# Agendeaki — Frontend

Interface web do projeto Agendeaki, desenvolvida com Next.js e TypeScript.

---

## 🛠️ Tecnologias

- [Next.js 16](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [ESLint](https://eslint.org/)

---

## ✅ Requisitos

- [Node.js](https://nodejs.org/) v18 ou superior
- npm v9 ou superior

---

## 🚀 Como rodar

### 1. Clone o repositório

```bash
git clone https://github.com/TCC-CAS/tcc-cas-2026-1-e-2-grupo-6.git
cd tcc-cas-2026-1-e-2-grupo-6/agendeaki/frontend
```

### 2. Instale as dependências

```bash
npm install
```

### 3. Rode em ambiente de desenvolvimento

```bash
npm run dev
```

### 4. Acesse no navegador

http://localhost:3000
